# Bootstrap 3 Official + Jade template w/ Gulpjs.

Compile with gulpjs and gulp-plugins. 

## Requirement

Bower.

## Usage

Install some components.

	$ npm install

Initialize

	$ gulp init

Run gulp

	$ gulp

or

	$ npm start
